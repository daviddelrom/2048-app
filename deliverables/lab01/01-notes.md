## Aprendizaje

- Utilización de Servidores MCP para tener información actualizada sobre las tecnologías y adoptar las mejores prácticas
- Descubrimiento del Servidor MCP Chrome Dev Tools
- Descubrimiento de simpleBrowser en vscode

## Metodología adoptada

- Utilización del agente "Plan" para diseñar la aplicación antes de implementarla
- Depuración de la aplicación via Chrome Dev Tools MCP para identificar y corregir errores

## Dificultades encontradas:

- Tailwind CSS: dificultad en configurar correctamente el archivo de configuración para que los estilos se aplicaran
  - Resuelto mediante la utilización de Nuxt UI MCP para tener la configuración correcta
- Error en la visualización de la puntuación
  - Resuelto utilizando Chrome Dev Tools MCP para inspeccionar los elementos e identificar el problema
